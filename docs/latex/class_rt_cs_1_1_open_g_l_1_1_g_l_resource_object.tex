\doxysubsection{Detailed Description}
Base class of \mbox{\hyperlink{namespace_rt_cs_1_1_open_g_l}{Open\+GL}} resourced object classes. 

\mbox{\hyperlink{namespace_rt_cs_1_1_open_g_l}{Open\+GL}} resourced object in this library means that object is managed by ID created by gl\+Gen\+X\+XX function. For example, buffer, texture and so on.
