Detailed Description

Base class of OpenGL resourced object classes.

OpenGL resourced object in this library means that object is managed by ID created by glGenXXX function. For example, buffer, texture and so on.

Call opengl-initialization method.

The class inherits GLResourceObject should be call initialize method in this method, for example, glGenTextures, glGenBuffers and so on.
This method will be called when opengl context is valid.
